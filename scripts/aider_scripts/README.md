# How to use

## Run single test

```bash
aider --no-gitignore --model <model_name> -f <prompt_file_path> <file1> <file2> <file...>
```

Example

```bash
aider --model o3 --no-gitignore -f ../aider_test/4713/4713.prompt src/function/table/read_csv.cpp
```
